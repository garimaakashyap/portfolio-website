ConstVidSearch - AI-Powered Video Search App

ConstVidSearch is a scalable Android application that enables real-time video search using transformer-based vector embeddings. It features constant-time retrieval by leveraging modern technologies such as Microsoft multilingual transformers, AWS (S3, DynamoDB), and Pinecone vector database. Key Highlights: Instant Search: Retrieves relevant videos in constant time using vector-based indexing. AI-Powered: Generates semantic embeddings from video titles and descriptions using multilingual transformers. Cloud-Integrated: Stores videos and metadata securely on AWS (S3 & DynamoDB). Smart Search Engine: Utilizes Pinecone for fast and accurate similarity search. User-Friendly: Clean interface for uploading and searching videos seamlessly.
